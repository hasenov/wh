Пошив купальника по вашим меркам

Каждая девушка хочет быть неотразимой на пляже, на волне или просто на отдыхe.
Но, выбрать купальник, который будет идеально сидеть по фигуре — задача не из простых.
Поэтому, мы всегда готовы сшить именно тот, идеальный купальник по вашим меркам.

При снятии замеров лента должна плотно прилегать по самым выступающим точкам*

Купальник без рукавов:

  1. Обхват груди
  2. Обхват талии
  3. Обхват бедер
  4. Длинна туловища (слитные модели)

Купальник с рукавами:

  1. Длинна рукава от запястья до плеча
  2. Обхват запястья
  3. Обхват бицепса
  4. Мерки для купальника без рукавов

Комбинезон:

  1. Обхват берда возле паховой области
  2. Обхват щиколотки
  3. Длину штанины по внутреннему шву
  4. Мерки для купальника без рукавов
  5. Мерки для купальника с рукавами
